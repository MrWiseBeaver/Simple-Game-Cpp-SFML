# Simple-Game-Cpp-SFML
Hi, this is my first game! <br>
I created this game at the beginning of my programming adventure. <br>
To run this game you have to just open .exe file. <br>
Hope you like it! :)

Key bindings:<br>
W - up<br>
A - left<br>
D - right<br>
Space - shoot<br>
Enter - run game
